How to compile:
1. Ensure you have the database created using ddl table and seeding files
2. Update database connection in main
3. Have postgresql-42.7.3.jar installed
4. Navigate to code in terminal then enter
javac *.java
5. Next enter (location may vary based on postgresql.jar version and location)
java -cp "C:/postgresql-42.7.3.jar;." Main
6. Test accordingly in terminal
